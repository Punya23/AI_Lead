Webhook Integration

You can test the system-to-system webhook ingestion by running this cURL command in your terminal. It returns a 202 Accepted immediately and processes asynchronously. 

curl -X POST http://localhost:8000/api/v1/webhooks/lead \
  -H "Content-Type: application/json" \
  -d '{
    "payload": {
      "name": "Priya Patel",
      "email": "priya.patel@example.in",
      "company": "Acme Corp",
      "inquiry": "We need help scaling our backend."
    },
    "source": "zapier"
  }'
